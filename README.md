# Rom Creative Toolkit

A clean, Apple-inspired browser toolkit for graphic designers and video editors. Most tools process files locally in the browser, helping keep creative files private.

## Included tools

- Image Converter — JPG, PNG, WEBP
- Image Compressor
- Resize & Crop with social presets
- Video to MP3
- ZIP File Compressor
- Image to PDF
- Social Media Size Presets
- Color Palette Extractor
- QR Code Generator
- Favorites, search, local history, Khmer/English UI
- White glass interface with optional dark mode

## Run locally

Open `index.html` in Chrome or Microsoft Edge. Video-to-MP3 needs internet access on first use to load FFmpeg WebAssembly.

## GitHub Pages

In the repository, open **Settings → Pages**, select **Deploy from a branch**, choose `main` and `/ (root)`, then save.

## Privacy

Image, ZIP, PDF, palette, QR, and browser-supported conversions are processed on the user's device. External JavaScript libraries are loaded from CDNs.
